- to initiate git use:
	git init
- the main branch which is master is created
- to add files to cache
	git add [.] - [file name]
- then to add to stagged files  
	git commit -m "[message]"
- to connect your local git repo to git remote
	git remote add origin [rep-url.git]
	git remote add origin2 [rep-url.git]
- to push your code to remote
	git push origin
	git push origin2
- to create new branch for development
	git branch [branch name]
